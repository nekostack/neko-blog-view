# neko-blog-view
近日在学习Node.js，欲利用Node.js & MongoDB编写一个博客系统（NekoBlog），从而熟悉MVC模式。

本repo为博客系统的view层模板，也是猫七的第一个仓库，目前已经逐渐熟悉了git的操作 ^ ^
